## 步骤

1. 前置操作：执行登录流程（参考 `auth/login.md`）
2. 导航到仪表盘首页 `{{BASE_URL}}/dashboard`
3. 等待页面完全加载（包括异步数据）
4. 截图记录仪表盘整体样式
5. 检查页面标题是否存在且非空
6. 检查页面上是否有可交互元素（按钮、链接等）
7. 检查左侧导航栏/侧边栏是否正常显示
8. 检查页面是否存在 JavaScript 控制台错误

## 预期结果

- 仪表盘页面正常加载，无白屏或加载失败
- 页面标题不为空
- 页面包含可交互元素
- 导航栏/侧边栏菜单项正常显示
- 无 JavaScript 控制台报错
- 各数据卡片/模块渲染完整，无空白区域（如有数据）
